# Collect Tools

A small tool for collecting data from setups and manipulating them.


## What's inside...

### Infrastructures

- collect_tools_main.py
- collect_tools_flows_list.py
- collect_tools_helper.py

### Flows

- certificates_by_roles_flow.py
- logs_by_roles_flow.py

### Scripts

- generate_excel_certificate.py
- generate_logs_by_roles_json.py

## How to run

    python2 ~/GIT/support/HealthChecks/tests/debug_tools/collect_tools/<script_name>.py
